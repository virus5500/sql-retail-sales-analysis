# Retail Sales & Customer Analysis (SQL)

This project focuses on analyzing retail sales data using SQL.
The dataset is stored in Excel and imported into a SQL database
for performing analytical queries.

## Dataset
- Customers
- Products
- Orders

## Key Analysis
- Total sales calculation
- Top selling products
- City-wise revenue analysis
- Repeat customer identification

## Tools Used
- SQL
- Excel

## Purpose
This project was created for learning and practice
of SQL data analysis concepts.
